# Speakers Directory

## Overview

The Speakers Directory is the main landing page showcasing all Zure speakers. It provides a browsable grid of speaker cards, allowing event organizers to discover speakers by browsing their profiles, expertise, and available sessions.

## Purpose

Help event organizers and companies:
- Discover Zure's expert speakers at a glance
- Browse speakers by photo, name, and title
- Quickly access individual speaker profiles for more details

## User Experience

### Layout
- **Grid of Speaker Cards**: Responsive grid displaying all speakers
- **Speaker Card Contents**:
  - Large speaker photo
  - Name (bold, prominent)
  - Title/role at Zure
  - Click anywhere on card to view full profile

### Interactions
- Click on any speaker card to navigate to their full profile page
- Hover states for visual feedback
- Mobile responsive: Grid adjusts from 3-4 columns on desktop to 1-2 on mobile

## Content Requirements

### Each Speaker Card Shows:
1. **Photo** - Professional headshot
2. **Name** - Speaker's full name
3. **Title** - Their role/title at Zure

### Visual Design
- Bold, high-contrast cards with clean borders
- Large, prominent photos (headshots)
- Black text on white cards for maximum readability
- Generous white space around each card

## No Search/Filter (for now)

Keep it simple - just a browsable grid. Future versions could add search and filtering by expertise area or topic.

## States

### Default State
- Grid of all speakers, sorted alphabetically or by featured status

### Empty State
- If no speakers exist: "No speakers yet" message

### Hover State
- Card slightly elevates or border becomes bolder
- Cursor changes to pointer indicating clickability

## Success Criteria

- Event organizers can quickly scan all available speakers
- Clear visual hierarchy makes names and titles easy to read
- Smooth, intuitive navigation to individual speaker profiles
